# API Escola

## Técnologias
- NodeJs
- Express
- Typeorm
- Postgres

## Requisitos
- Docker composer

## Preparando ambiente

- Clone o repositório
- Renomei o arquivo `.env.exemple` para `.env`
- Configure as variavei de acordo com sua preferencia

## Inciando a api
- Rode `docker-compose up --build`
- Rode `docker exec node yarn typeorm migration:run`
